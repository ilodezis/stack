# 💊 Стек добавок — Персональный трекер (PWA)

Красивое, быстрое и полностью приватное веб-приложение для удобного трекинга приема витаминов, минералов и добавок непосредственно с вашего мобильного телефона.

Приложение спроектировано как **Progressive Web App (PWA)** — его можно установить на главный экран смартфона, и оно будет полноценно работать без доступа к интернету.

---

## ✨ Ключевые возможности

*   **📱 Мобильный PWA-интерфейс**: Устанавливается как нативное приложение на iOS и Android. Работает мгновенно и полностью автономно (офлайн).
*   **🎨 Премиум-дизайн**: Современный интерфейс со стеклянными эффектами (glassmorphism), плавными анимациями, а также автоматической и ручной поддержкой темной/светлой темы.
*   **🔄 Интерактивный Drag & Drop**:
    *   Смена порядка карточек времени суток.
    *   Упорядочивание добавок внутри блоков.
    *   Перетаскивание добавок *между* разными блоками (например, перенос из «Утро» в «День»).
*   **⚙️ Полная кастомизация**: 
    *   Редактирование, добавление и удаление блоков (можно настраивать названия, подзаголовки, emojis и цветовые темы).
    *   Редактирование названий, условий приема и дозировок конкретных добавок.
*   **⏱️ Ежедневный трекинг приема (отключаемый)**:
    *   Возможность отмечать галочками принятые сегодня добавки.
    *   Анимированное вычеркивание и прогресс-бар дня.
    *   Автоматический сброс прогресса при наступлении нового дня.
    *   *Функцию можно полностью отключить в настройках, вернув чистый статический список.*
*   **🔒 100% Конфиденциальность**: Данные хранятся исключительно в памяти вашего браузера (`localStorage`). У проекта нет бэкенда и баз данных — никто посторонний не может увидеть ваш список.
*   **💾 Резервное копирование**: Быстрый экспорт всего стека в JSON-файл и импорт настроек на другом устройстве.

## 🚀 Как установить на телефон

1.  Откройте ссылку на веб-приложение в браузере вашего смартфона.
2.  **На iOS (Safari)**: Нажмите кнопку **«Поделиться»** (Share) -> выберите **«На экран "Домой"»** (Add to Home Screen).
3.  **На Android (Chrome)**: Нажмите на три точки меню -> выберите **«Установить приложение»** или **«Добавить на главный экран»**.
4.  На рабочем столе появится иконка «Стек добавок», открывающая приложение в полноэкранном режиме без рамок браузера.

## 🛠️ Локальная разработка

Проект написан на чистом стеке веб-технологий (Vanilla HTML, CSS, JS) и не требует этапа компиляции или сборки.

Для запуска локально достаточно запустить любой статический веб-сервер в корневой папке проекта.

### Использование Python:
```bash
python -m http.server 8000
```
После этого откройте в браузере: `http://localhost:8000`

---

## 🧬 Технологический стек

*   **HTML5 & CSS3**: Семантическая верстка, CSS Custom Properties (переменные) для темизации, CSS Grid & Flexbox.
*   **JavaScript (ES6+)**: Модульная архитектура, события DOM, работа с LocalStorage и File API.
*   **SortableJS**: Библиотека для обеспечения плавного, отзывчивого перетаскивания элементов на сенсорных экранах.
*   **PWA**: Web App Manifest (`manifest.json`) и Service Worker (`sw.js`) со стратегией кэширования *Stale-While-Revalidate*.
